CSS Must Knows

Specificity example:

<html>
  <link rel="stylesheet" href="main.css">

    <div>
      <h1 id="id-style" class="class-style">Hi</h1>
    </div>
  </body>
</html>


h1 {
  background: crimson;
}
.class-style {
  background: violet;
}

  background: dodgerblue;
